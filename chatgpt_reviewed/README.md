# ChatGPT Reviewed

Store article-level Markdown files returned from ChatGPT web review here before final approval.

Place full article replacements in:

- `articles/`

Recommended filenames:

- `005-topic-slug.md`
- `topic-slug.md`

Workflow role:

- `generated/` contains draft articles and queue outputs
- `chatgpt_reviewed/articles/` contains ChatGPT-reviewed full replacements
- `approved/articles/` contains only the final accepted versions ready for merge-back

Use this folder as the handoff destination for the ChatGPT web review phase.
