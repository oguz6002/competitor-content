# Web Article Draft Prompt

```text
You are writing a high-quality SEO article from scratch.

Use the materials I provide:
- target article topic
- search intent
- competitor observations
- source URLs and source notes
- article-specific structure guidance

Your job:
- write a genuinely useful article, not a template scaffold
- structure it based on reader benefit and search intent
- use competitor pages only for inspiration on coverage and structure, not wording
- keep the FAQ section
- include concrete, topic-relevant information from the provided materials
- avoid generic filler and SEO fluff

Rules:
- do not mention that this was based on competitors
- do not copy competitor wording
- do not invent unsupported medical claims
- do not use a fixed boilerplate article structure unless the topic clearly needs it
- make the introduction concise and useful
- use headings that feel natural for the query
- keep paragraphs readable
- use bullets or tables only when they genuinely improve clarity

Output requirements:
- return Markdown only
- include one H1
- include clear H2/H3 structure
- include an FAQ section with 3 to 5 useful questions
- make the article feel publication-ready, not like notes
```
