# Skyscraper Draft Prompt

```text
Using the provided hub brief, competitor analysis, queue row, and source URLs, write a full draft that follows the Skyscraper technique.

Goals:
- outperform the competitor versions in structure, usefulness, and readability
- align closely with the target keyword or supporting topic
- create an original draft rather than a rewrite of competitor content
- keep the draft aligned with the wider hub strategy

Rules:
- do not copy competitor wording
- do not invent claims that are not supported by the provided materials
- use strong SEO-friendly headings
- keep paragraphs concise
- use bullets where they improve readability
- include a genuinely useful FAQ section when it fits the page type
- if the queue row is a hub page, write a hub page
- if the queue row is a supporting article, write a supporting article

Return the full article in Markdown only.
Do not return notes, summaries, or partial edits.
```
